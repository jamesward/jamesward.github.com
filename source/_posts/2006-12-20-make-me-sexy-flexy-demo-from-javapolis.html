My "Flex and Java" session on Friday was tons of fun. In a few minutes I was able to build "Sexy Flexy", an application with a Sexy Flex front-end and a simple Hibernate back-end. The application allowed users to take their picture, enter their name, and save them to the database. Then using Flex Data Services these pictures push in real-time out to all of the people using the application. I was hoping that everyone in the audience with a web cam would get on my wireless network and use the application... True audience participation! Unfortunately my wireless router was not cooperating, so it was just Christoph Rooms and I taking pictures. Oh-well. But I have posted the application (slightly modified). Check it out: Make me Sexy Flexy! If you are interested in seeing the code, it's on SourceForge.
